# RUNWAY 10 Design QA

## Direction

1990s corporate crisis room crossed with a printed board dossier. Near-black green field, acid status color, warning red, editorial serif headlines, and mono operational labels form one deliberate system.

## Verified surfaces

- Desktop briefing: strong product identity, daily seed, localized language controls, confidential rules dossier.
- Desktop decision room: ten-step progress, live company metrics, two explicit irreversible choices, consequence report.
- Mobile briefing at 390 × 844: no horizontal overflow; content and dossier remain readable in one vertical flow.
- Reduced-motion media query removes nonessential animation.
- Keyboard focus uses a visible high-contrast outline inherited from global controls.

## Runtime evidence

- Browser content check: passed.
- Next.js error overlay: absent.
- Browser page errors: none.
- Desktop and mobile screenshots reviewed during implementation.
- Playwright covers localization, decision effects, local persistence, mobile controls, and legacy-route redirects.
