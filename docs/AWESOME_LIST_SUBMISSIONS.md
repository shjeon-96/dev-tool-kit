## Tips for Approval

1. **Follow Guidelines**: Read CONTRIBUTING.md carefully
2. **Quality Check**: Ensure link works, description is accurate
3. **Alphabetical Order**: Most lists require alphabetical sorting
4. **Concise Description**: Keep it short but informative
5. **Wait Time**: Allow 1-2 weeks for review
6. **One PR at a Time**: Don't spam multiple lists simultaneously

---

## Tracking

After submission, update the status:

- `Pending` - PR not yet submitted
- `Submitted` - PR created, awaiting review
- `Approved` - PR merged
- `Rejected` - PR closed (with reason)
