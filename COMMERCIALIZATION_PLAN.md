## 1. 현재 판단

RUNWAY 10은 이제 텍스트 TRPG나 양자택일 퀴즈가 아니다. 사무실 보드, 부서 캐릭터, 3장 액션 손패, CEO 성향, 부서 연쇄, 월말 사건, 6개월 정산을 가진 실제 플레이 가능한 경영 로그라이크 버티컬 슬라이스다.

다만 **상업성이 입증된 게임은 아니다.** 현재 확보한 것은 판매 가능한 게임의 첫 검증 빌드다. 재미, 재방문, 공유, 지불 의사는 실제 사용자 데이터로만 판정한다.

## 2. 제품 정의

- 이름: `RUNWAY 10`
- 장르: 5분 일일 경영 카드 로그라이크
- 핵심 판타지: 제한된 현금으로 팀을 배치하고 6개월 뒤 데모데이에 도달한다.
- 대상: 스타트업·직장·경영 콘텐츠에 익숙한 20~40대 웹 사용자
- 세션: 회사/CEO/8장 덱 선택 → 매월 3장 중 1장 실행 → 사무실 변화/생산/사건 정산 → 데모데이 → 재도전/공유
- 일일성: UTC 날짜와 업종으로 손패와 사건이 결정된다.
- 실패: 현금 0, 사기 0, 신뢰 0
- 승리: 6개월 생존
- 재도전: 같은 일일 시드를 다시 풀어 학습한다.

## 5. 상업성 가설

### 무료 웹 모델

일일 무료 게임으로 공유와 재방문을 만들고, 종료 화면에만 광고를 둔다. Google은 게임 조작부와 광고 사이에 충분한 거리를 두고 오클릭을 피하라고 안내한다. 현재 광고는 플레이 중 노출하지 않고 결과 흐름과 분리한다.

### 프리미엄 확장 모델

유지율이 합격한 뒤에만 아래를 검토한다.

1. `RUNWAY 10: Founder Edition` 유료판: 캠페인, 더 긴 런, 직원/프로젝트 메타
2. Steam 데모: 웹 빌드를 무료 데모로 사용하고 위시리스트 수요 검증
3. B2B 시즌 스폰서: 게임 효과나 점수에는 관여하지 않는 결과/시즌 브랜딩

광고 제거 구독, 가챠, 에너지, 보상형 광고는 현재 범위가 아니다.

## 6. 합격 지표

최소 신규 사용자 500명, 최소 14일을 확보한 뒤 평가한다.

| 지표                    | 실패 |   관찰 | 합격 |
| ----------------------- | ---: | -----: | ---: |
| 로비→첫 카드 실행       | <45% | 45~65% | ≥65% |
| 시작→6개월 종료         | <35% | 35~55% | ≥55% |
| 종료→즉시 재도전        | <12% | 12~25% | ≥25% |
| 정확한 D1               | <15% | 15~25% | ≥25% |
| 정확한 D7               |  <4% |   4~8% |  ≥8% |
| 공유 완료               |  <2% |   2~5% |  ≥5% |
| 3일 이상 주간 반복 완료 |  <8% |  8~15% | ≥15% |

### 돈을 받기 전 게이트

- 2주 연속 주간 완료 사용자 5,000명
- 완료율 55% 이상
- D1 25% 이상, D7 8% 이상
- 즉시 재도전 25% 이상
- 공유 완료 5% 이상
- 광고 정책/무효 트래픽 경고 0
- 10명 정성 테스트 중 7명 이상이 “다른 전략으로 다시 하고 싶다”고 응답

미달 시 광고를 늘리지 않는다. 카드 읽기, 선택 결과 가시성, 전략 다양성, 세션 리듬을 먼저 수정한다.

## 8. 다음 제작 순서

### Phase 1 — 첫 60초 개선

1. 카드 hover/focus에 정산 예상치 표시
2. 첫 판만 3단계 튜토리얼 오버레이
3. 카드 실행을 클릭이 아닌 보드 부서 슬롯 배치로 강화
4. 월 정산 애니메이션을 비용→카드→사건→연쇄 순으로 표현
5. 5명 관찰 테스트에서 규칙 설명 없이 플레이 가능한지 확인

완료 조건: 첫 카드 실행률 65%, “무엇을 눌러야 하는지 몰랐다” 0/5.

### Phase 2 — 전략 깊이

1. 현재 직원 4명을 8명으로 확장: 고유 능력과 피로
2. 현재 프로젝트 5개를 12개로 확장: 요구 부서 레벨과 상호 배타 선택
3. 현재 연쇄 사건 6개를 18개로 확장
4. 카드 30개로 확장하되 역할이 겹치는 카드는 만들지 않음
5. 승리 엔딩 4개: 제품, 매출, 신뢰, 균형

완료 조건: 10회 테스트에서 동일한 최적 카드 반복률 35% 미만.

### Phase 3 — 반복성

1. 실패 후 영구 해금은 카드/CEO/사무실 장식 중 하나만 보상
2. 일일 도전과 자유 시드 분리
3. 주간 리그는 로그인 없이 익명 ID로 시작
4. 공유 카드에 엔딩과 사무실 상태를 반영

완료 조건: D1 25%, D7 8%, 주간 3일 반복 완료 15%.

### Phase 4 — 상업화 실험

1. 종료 광고 RPM/이탈 측정
2. 스폰서 없는 기본판과 시즌 브랜딩 목업 비교
3. 프리미엄 캠페인 랜딩과 가격 의향 조사
4. Steam Coming Soon은 아트 방향과 핵심 기능이 고정된 뒤 개설
5. 위시리스트와 웹 D7이 모두 합격할 때만 유료 제작 확정

## 9. 광고 안전 규칙

- 플레이 보드와 카드 손패 안에 광고를 넣지 않는다.
- 광고와 재시도/공유/다음 버튼 사이 150px 이상 유지한다.
- 광고를 카드, 보상, 스폰서 선택처럼 보이게 만들지 않는다.
- 광고 시청을 해금이나 점수와 연결하지 않는다.
- 자체 테스트에서 광고를 클릭하지 않는다.
- 정책 위반 또는 무효 트래픽 경고가 있으면 광고 요청을 중지하고 원인을 수정한다.
